# Trader demonstration script

Aim to run this in under five minutes and ask operational questions afterwards.

1. Open `/brightwash` on a phone-sized screen. Explain that this could be linked from WhatsApp, Facebook or a Google profile.
2. Complete the structured enquiry: driveway, condition, add-on and photo step.
3. Switch to the trader view from the confirmation page.
4. Show the organised job detail and the **Brenda** summary panel. State that it is advisory only.
5. Open **Create quote**, adjust any line item or deposit, and preview it.
6. Show the customer quote view and the deposit requirement.
7. Simulate payment, then return to the job record showing **Deposit Paid**.

## Questions to ask

- Would this capture enough information for you to decide whether to quote remotely?
- Which fields are missing or unnecessary?
- Which add-ons should be available by default?
- Do you ask for deposits; fixed amount or percentage?
- Would you send customers to a page like this?
- Would you trial it with real enquiries after security and payment controls are in place?
